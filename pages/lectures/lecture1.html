Estructura básica de un Sitio WEB

Actualmente cualquier sitio web inicia con la apertura de la etiqueta HTML que en su interior contiene a la etiqueta Head para la información técnica del navegador y a la etiqueta Body para el contenido de la página Web.


Etiquetas de Contenido

HTML5 hace uso de 5 etiquetas que permiten estructurar el contenido que queremos mostrar en nuestra página Web, estas etiquetas son:

header : El encabezado de nuestro sitio Web.
nav : El menú de navegación para nuestro sitio web.
aside: Un menú lateral que muestra información relevante como las ultimas noticias de nuestro sitio web.
section: Secciones de contenido de nuestro sitio web que se puede dividir con la etiqueta article.
footer: Pie de página de nuestro sitio web.
