Lectura 2

Presentación

En esta sección vamos a aprender la aplicación básica de estilos css a través de selectores y clases.


¿Que es un selector CSS?

El selector CSS es el identificador en la hoja de estilos del elemento o elementos del DOM a los que se aplicarán las declaraciones contenidas en la regla de la que forma parte dicho selector.


MDN, Contributors. (2019) Ilustración de la estructura Básica CSS. [Figura] Recuperado de https://developer.mozilla.org/es/docs/Learn/Getting_started_with_the_web/CSS_basics

¿Que es una clase CSS?

En un documento HTML, los selectores de clase buscan un elemento basado en el contenido de su atributo class. El atributo class está definido como una lista de elementos separados por espacio, y uno de esos elementos debe coincidir exactamente con el nombre de clase dado en el selector.


Aquino, C. (2015) Ilustración de clases CSS. [Figura] Recuperado de https://www.slideshare.net/cesarjam/clases-css-en-wordpress

Propiedades Básicas

  • Width: Ancho de un elemento
  • Height: Alto de un elemento
  • Margin: Margen de un elemento
  • Padding: Margen interior de un elemento
  • Color: Color del texto y bordes
  • Background-color:Color del fondo
